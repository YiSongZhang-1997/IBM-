# IBM 推荐系统 - Udacity NanoDegree Project

## 项目简介
分析用户与 IBM Watson Studio 平台上的文章进行的互动，向他们推荐他们可能会喜欢的文章。创建一个推荐系统，向特定的用户显示最相关的文章。
